# Bot_Discord
Bot discord développé uniquement en Python

Vous pouvez set up le prefix ici : client = commands.Bot(command_prefix = '=',intents=intents)
Vous devez activer les intents dans les options du bot discord sur la plateforme de développement de discord
N'oubliez pas de mettre votre token dans les guillemets tout en bas dans le client.run("")
Pour le lancer grâce à votre terminal, il vous faudra Python3 et taper la commande : python3 bot.py quand vous êtes dans le bon fichier et le bot devrait vous envoyer un "connected" dans le terminal quand celui-ci est connecté
